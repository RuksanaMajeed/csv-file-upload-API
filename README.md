## Installation

Follow these steps to set up the project locally.

1. **Clone the repository:**

    ```sh
    git clone https://github.com/RuksanaMajeed/csv-file-upload-API.git
    cd csv-file-upload-API
    ```

2. **Install dependencies:**

    ```sh
    npm install
    ```

## Usage

To start the server:

```sh
npm start
```
